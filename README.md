## 📂 Installation & Setup

1. Clone the repository:
   ```sh
   git clone https://github.com/your-username/your-portfolio.git
   ```
2. Navigate to the project folder:
   ```sh
   cd your-portfolio
   ```
3. Install dependencies:
   ```sh
   npm install
   ```
4. Run the development server:
   ```sh
   npm run dev
   ```
5. Open your browser and visit:
   ```
   http://localhost:3000
   ```
